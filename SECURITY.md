# Security Policy

## Supported Versions

To ensure the security of your site, please keep the plugin updated. We actively support the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| 1.1.x   | :x:                |
| < 1.1   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please do **not** open a public issue.

Instead, please report it directly via:

- **Email**: security@sldevs.com
- **GitHub**: Private vulnerability reporting (if enabled on this repository)

We will respond within 48 hours to acknowledge the report and provide a timeline for remediation.

### Process

1. **Disclosure**: You report the vulnerability privately.
2. **Investigation**: We confirm the issue and assess its severity.
3. **Fix**: We develop and test a patch.
4. **Release**: We ship a security update (e.g., v1.2.2).
5. **Announcement**: We disclose the vulnerability details after a safe period to allow users to upgrade.

Thank you for helping keep True Video Product Gallery secure!
